# Sovereign Prompt Protocols
## Directory Intent & Context
The operational core of the repository's interaction layer. This directory holds highly sophisticated, context-dense system prompts, cognitive scaffolding directives, and architectural blueprints for LLMs.

## Contents
- **System Prompts:** Foundational directives setting the identity, bounds, and reasoning logic for agents (e.g., `🜂 ATLAS IMAGING ARCHITECT - SYSTEM`, `COGNITIVE SCAFFOLD`).
- **Protocols:** Procedural prompts like the `Architecture-as-Oracle Protocol (AAO-P)` and the `Contextual Topological Anchor (CTA) Protocol`.

## Cartographic Significance
These prompts are not mere requests; they are software written in natural language. They are the linguistic enforcement of the architectural topologies defined in `Architectures/` and `AgentProfiles/`.
