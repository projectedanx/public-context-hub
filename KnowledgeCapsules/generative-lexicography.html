Generative Lexicography: The Formal Language of AI Creation

Generative Lexicography is the study of specialized communication between humans and generative models. It shifts the perspective of 'prompt engineering' from a collection of tricks to a formal system that targets the probabilistic and structural logic of the machine's internal world model. By understanding the diffusion process as a foundational grammar, users can navigate the latent space with mathematical and linguistic precision.

Core Thesis

From Natural Language to Machine Command

Effective interaction with AI requires a hybrid, formal-declarative language. This system utilizes a bipartite structure—the 'Imaginative Core' for creative intent and the 'Technical Stack' for execution parameters—to provide a coordinate system for the high-dimensional latent space.

Foundational Pillars

These concepts form the technical and philosophical bedrock of the generative language.

Denoising Diffusion Grammar

The temporal logic of image generation follows a hierarchical sequence: early timesteps establish low-frequency foundational information (composition/layout), while later steps refine high-frequency details (textures/fine features). Prompts must align with this timeline to be effective.

Pustejovsky’s Qualia Framework

A complete semantic prompt addresses four roles: Formal (what it is), Constitutive (what it's made of), Telic (what it's for), and Agentive (how it came to be). Semantic incompleteness in any role leads to model-biased hallucinations.

Model Dialects

Generative models exhibit distinct interpretive philosophies. Midjourney favors 'artistic bias' and cinematic feel, whereas DALL-E 3 prioritizes 'literalism' and instruction-following. Expert lexicography requires multilingual proficiency across these model-specific dialects.

The Technical Stack

A specialized lexicon of computer graphics, optics, and cinematography jargon. Terms like 'raytraced microfacet BRDF' or '50mm lens' act as high-efficiency tokens that target pre-computed regions of the model's possibility space.

The Generative Pipeline

The operational process of translating intent into high-fidelity output.

Step 1

Semantic Framing

Define the Imaginative Core using Pustejovsky's Qualia roles to ensure the model has a complete definition of the target subject.

  • •Establish the Formal category (e.g., 'a portrait').
  • •Define Constitutive materials (e.g., 'polished chrome').
Step 2

Technical Layering

Apply the Technical Stack to define execution parameters such as lighting, camera optics, and rendering style.

  • •Select lighting models (e.g., 'Global Illumination').
  • •Specify virtual camera specs (e.g., 'f/1.8 aperture').
Step 3

Syntactic Modulation

Use weighting, negative prompts, and delimiters to fine-tune the influence of specific tokens on the denoising process.

  • •Assign numerical weights (e.g., ':1.3').
  • •Exclude failure modes via negative prompting.

System Vulnerabilities

Risks inherent in high-level human-AI linguistic interaction.

Cargo Culting

Using technical terms as empty tokens rather than functional instructions.

Mitigation Strategy

  • →Cross-referencing terminology with CG documentation
  • →Verifying token effects via A/B testing

Semantic Drift

The tendency of words to change meaning between different model versions or platforms.

Mitigation Strategy

  • →Maintaining model-specific dictionaries
  • →Using standardized UGL protocols where possible

Mode Collapse

A failure in GAN-based models to produce diverse outputs.

Mitigation Strategy

  • →Utilizing Diffusion-based models for more stable results
  • →Varying the random noise seed

Semantic Coherence

The logical consistency of the generated world model.

Absence of hallucinated limbs Physics-compliant lighting shadows

Prompt Engineering Audit

An actionable guide for practitioners to ensure precision in their commands.

Semantic Completeness Check

  • Does the prompt define the subject's origin (Agentive)?
  • Is the purpose or context defined (Telic)?
  • Are the materials and parts explicitly stated (Constitutive)?

Structural Alignment Check

  • Is the prompt separated into Core vs. Technical Stack?
  • Are compositional cues prioritized in the phrasing?
  • Are technical tokens specific enough for the target model?

Dialect Optimization

  • If using Midjourney, are aesthetic modifiers included?
  • If using DALL-E, is the instruction literal and descriptive?
  • Have negative tokens been added for common failure modes?
