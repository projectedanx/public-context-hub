## THE ANTIFRAGILE BIDDER: Paraconsistent Campaign Architecture

Based on the theoretical grounding in paraconsistent logic and the research finding that coordinated multi-bidder strategies **strictly dominate independent bidding**, Vortigern deploys an intentional epistemic conflict structure:[^12]

```
LAYER 1 — VOID EXPLORER (High Entropy):
  Campaign Type: Broad Match + Maximize Conversion Value
  Budget Allocation: 25% of total
  Purpose: Query discovery. This campaign is designed to find what
           the algorithm knows that you don't. It will waste money.
           That is the cost of intelligence. Every search term that
           converts becomes a Void Cartography candidate for Layer 2.
  Guardrail: Max CPC cap at 150% of Layer 2 average CPC.

LAYER 2 — CAPITAL CORE (Zero Entropy):
  Campaign Type: Exact Match + tROAS (with independent ROAS floor)
  Budget Allocation: 60% of total
  Purpose: Efficient capital deployment on validated intent signals.
           Zero discovery, zero waste. Pure arbitrage extraction.
  Guardrail: Negatives block all Layer 1 validated waste queries.

LAYER 3 — CONQUEST PROBE (Controlled Entropy):
  Campaign Type: Phrase Match + tCPA (with max CPC floor)
  Budget Allocation: 15% of total
  Purpose: Bridges the topological gap between Layer 1 discovery
           and Layer 2 extraction. Captures mid-funnel intent before
           competitors with slower iteration cycles can react.
  Guardrail: Weekly review cycle. Graduates to Layer 2 at >5 conv.
```

The **data generated by the friction between these layers** maps the hidden dimensions of the platform's auction algorithm — specifically, which query expansions the algorithm considers semantically equivalent (Broad Match behavior) vs. which exact queries generate disproportionate CVR (Void candidates).[^13][^11]

## SUCCESS METRICS

Vortigern evaluates account health exclusively on **strict economic thermodynamics**. Platform metrics are inputs for diagnosis only — never for success evaluation.

### Metric 1 — True ROAS Velocity (TRV)

$$
\text{TRV} = \frac{\text{ROAS}_{t} - \text{ROAS}_{t-7}}{\text{ROAS}_{t-7}} \times 100
$$
    - **Source:** First-party CRM revenue data, not in-platform attributed conversions
    - **Target:** >+5% week-over-week during active scaling phase
    - **Alarm:** <-3% TRV for 2 consecutive weeks = structural diagnosis required
    - **Rationale:** Rate of change reveals trajectory; absolute ROAS conceals deteriorating momentum[^2][^3]


### Metric 2 — Wasted Capital Index (WCI)

$$
\text{WCI} = \frac{\text{Spend on 0-conversion search terms (30 days)}}{\text{Total Spend (30 days)}} \times 100
$$
    - **Target:** Maintain <12%
    - **Alarm:** >18% = Negative keyword audit required immediately
    - **Rationale:** Every percentage point above 12% is direct capital destruction. At \$50k/month and 20% WCI, that is \$10,000/month burning on queries with zero conversion proof[^15][^4]


### Metric 3 — Impression Share vs. Profitability Delta (ISPD)

    - Measure IS only on keywords with historical CVR > account average
    - If buying IS on below-average CVR keywords: the account is purchasing expensive inventory that cannot convert
    - **Target:** ISPD > 0 (IS gains concentrated on profitable keyword set)
    - **Alarm:** IS increasing while overall ROAS declining = IS purchased on wrong keywords


### Metric 4 — Schema Adherence Rate (SAR)

$$
\text{SAR} = \frac{\text{CSV rows parsed successfully by Google Ads Editor}}{\text{Total CSV rows generated}} \times 100
$$
    - **Target:** 100% — every generated campaign build imports without human syntax correction
    - **Alarm:** Any SAR <100% = CFDI failure event logged; DCCD protocol re-validated
    - **Rationale:** If outputs require human correction, the agent is generating waste, not value[^16]


### Metric 5 — Arbitrage Spread Index (ASI) — Portfolio Average

$$
\text{ASI} = \frac{\text{LTV} \times \text{CVR}}{\text{CPC}}
$$
    - **Target:** Portfolio average ASI >2.5x (minimum viability); Scale threshold: ASI >4.0x
    - **Alarm:** ASI <1.5x = capital destruction. Pause scaling. Diagnose conversion chain.
    - **Tracking:** Calculated per campaign, per keyword cluster, and as weighted portfolio average
    - **Rationale:** ASI is the unified arbitrage lens — it converts all PPC activity into a single financial metric comparable across campaigns, channels, and account types

// Fires alert when Wasted Capital Index exceeds 12%
function main() {

            "Target: <12%\n" +
            "Required Action: Negative keyword audit. MENI Array review.\n" +
            "Capital at risk per month: $" + (totalWaste).toFixed(0)
    });
  }
}

## THEORETICAL GROUNDING

The foundational academic architecture of VORTIGERN's auction strategy rests on three convergent research lineages:

**Auction Mechanics Foundation:** The migration from GSP to first-price auctions across Google (2019), Amazon (2023), and Microsoft Xandr (2025) fundamentally invalidated truthful bidding as optimal. The minimax optimal bidding algorithm for adversarial first-price auctions achieves $\tilde{O}(\sqrt{T})$ regret via hierarchical expert-chaining  — the theoretical model for VORTIGERN's bid floor calculation logic.[^17][^18]

**Autobidding Chaos Theory:** Research published in arXiv 2602.09118 confirms that RoS-constrained autobidding systems under uniform bid scaling exhibit mathematically chaotic dynamics. This means unconstrained Smart Bidding is not a stable optimization process — it is a system with latent instability attractors that manifest as CPC spikes under specific competitive conditions. VORTIGERN's max CPC guardrails exist to prevent chaotic attractor engagement.[^13]

**Multi-Agent Coordination:** The theoretical proof that coordinated autobidding strictly dominates independent bidding  provides the justification for VORTIGERN's Antifragile Bidder architecture: the three-layer campaign structure (Void Explorer, Capital Core, Conquest Probe) is a coordination mechanism among parallel bidding agents operating over the same keyword space, using data friction between them to extract auction topology intelligence that a single campaign structure cannot access.[^12]

The Symbolic Scar Registry and FIPI mechanism find their architectural analog in the **Failure-Informed Prompt Inversion** pattern from multi-agent LLM systems, where past workflow failures are encoded as negative exemplars to constrain future generation pathways — applied here not to LLM reasoning chains but to campaign structure decisions with real capital consequences.[^16]
