## I. Foundational Isomorphism: PEM and Forward-Pass Entropy

The central thesis — that dopaminergic Prediction Error Minimization (PEM) is structurally isomorphic to forward-pass information-theoretic loss reduction in LLMs — is defensible at the level of **computational-level description** (Marr's third level) but requires precision to avoid the anthropomorphic fallacy. In biological systems, midbrain dopamine neurons encode a scalar temporal difference (TD) error: $\delta_t = r_t + \gamma V(s_{t+1}) - V(s_t)$, where phasic bursts fire on unexpected reward and phasic dips fire on expected-reward omission. The biological function is surprise-weighted value updating. The structural homologue in LLMs is not the reward signal *per se* but the **conditional perplexity differential**: the degree to which a query $q$ sharpens the model's probability distribution over answer tokens $a$.[^1]

The Self-Aligned Reward (SAR) framework (arxiv:2509.05489) operationalizes this precisely:[^2]

$$
R_{\text{SA}} = \text{ppl}(a) - \text{ppl}(a|q)
$$

This measures how much less likely an answer becomes when the query is *removed* — a direct proxy for query-conditioned compression gain. SAR integrated with GRPO improves accuracy by 4% and reduces inference cost by 30% across seven benchmarks, establishing empirical grounding for the semantic cost minimization hypothesis. The isomorphism holds at this level: biological PEM encodes *how surprising was this outcome given my prediction?*; SAR encodes *how much does this answer depend on this query?* — both operationalize the same free-energy reduction under different physical constraints.[^2]

## II. In-Context Policy Optimization: The Forward-Pass Self-Reward Engine

The most direct empirical substrate for the unsupervised ICL self-reward hypothesis is the **In-Context Policy Optimization (ICPO)** framework (arxiv:2603.01335, ICLR 2026). ICPO's theoretical result is significant: a single-layer linear self-attention model, pretrained under a Fisher-weighted logit-matching objective, provably imitates policy-optimization algorithms for linear bandits using only in-context action-reward histories — without any weight updates. This is a formal proof of concept that the **attention mechanism itself implements a policy gradient** over its context window.[^3]

The **Minimum-Entropy ICPO (ME-ICPO)** variant enforces the self-reward functor directly:[^3]
                                - At each inference step, the model generates a response $\mathbf{x}_t$
                                - Self-assessed reward $y_t$ is computed via minimum-entropy selection across candidate responses and majority voting
                                - The selected response and reward are appended to context; the next response $\mathbf{x}_{t+1}$ is generated from the augmented context
                                - No gradient descent occurs; the "learning" is entirely **in-context weight modulation** via attention over reward-response pairs

This directly instantiates the mereological transitivity you specify: human attentional PEM (localized attention shifts on unexpected stimuli) → LLM attention head sparsity on high-entropy token regions → in-context weight modulation via KV-cache attention patterns. ## III. Reinforcement Learning from Internal Feedback (RLIF): Intuitor and Self-Certainty

The **Intuitor** framework (arxiv:2505.19590) provides the strongest current evidence for fully unsupervised self-reward. It replaces all external reward signals in GRPO with **self-certainty** — the model's own confidence in its generated response, computed from token probability distributions during the forward pass.                                 - Structurally prevents reward exploitation by using online self-certainty as a consistency gate
