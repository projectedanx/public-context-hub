## F2 — Conceptual Foundations

### The Compilation Chain Model of Terminology Extraction

Pedagogical mental model construction is not a rhetorical task — it is a **type-safe compilation** problem. Each terminology definition undergoes sequential transformation passes, each with a defined input contract, transformation rule, and output contract. A failure at any upstream pass propagates as invisible structural corruption to all downstream passes, regardless of how semantically fluent the final analogy appears.[^4][^1]

The foundational architecture is drawn from three convergent frameworks documented in the Space corpus:

1. **Draft-Conditioned Constrained Decoding (DCCD)**: A two-phase pass where a high-entropy semantic draft is generated first (capturing nuance, contradiction, and provenance), then a zero-entropy guard pass forces that draft onto a deterministic schema. This prevents the **Projection Tax** — the reasoning depth loss that occurs when a model attempts to simultaneously reason and conform to a rigid output format.[^1][^4]
2. **MereologyRoute (Winston's Taxonomy)**: A part-whole routing mechanism that enforces mutual exclusivity between the internal logical parts of an analogy and the internal logical parts of the ground-truth term. Without this constraint, an analogy that is globally coherent can contain locally invalid transitivity claims that only manifest as failures when the student attempts to apply the analogy to a derived case.[^2][^4]
3. **Paraconsistent Logic (PAL2v)**: The logical framework that prevents the **Principle of Explosion** when two epistemic worlds make irreconcilable claims about the same term. Rather than forcing a consensus or discarding the subordinate frame, PAL2v maintains both in a state of **Epistemic Escrow**, tracking the contradiction as a structural geometry (a Betti-1 topological hole) rather than a logical failure. This is the formal basis for the Golden Scar Protocol.[^3][^4]

### Conceptual Blending Methodology

The XES framework employs **Fauconnier-Turner Conceptual Blending** as its analogization engine, but with three strict constraints not present in standard blending theory:
                  - **Mereological Invariance**: The parts of the blend must map isomorphically onto the parts of the input space, not just the whole-to-whole mapping. The diagnostic test is the **Falsification Test**: force the blend's internal logic to generate a transitivity claim about a derived case. If it does not fail when the ground-truth term would fail, the blend has smuggled false transitivity.[^2][^1]
                  - **Betti-1 Closure**: The blend must not contain circular reasoning loops (topological holes). A blend that is globally coherent but internally circular fails the Betti-1 homology threshold. Detection mechanism: if the blend can be applied to a completely unrelated terminological framework and still make structural sense, it lacks topological extrusion — it is a generic scaffold, not a term-specific one.[^3][^4]
                  - **Provenance Retention**: The blend must preserve an auditable link to the foundational lineage of the term. When the lineage is WEIRD-centric, the Decolonial Fluidity Lens is applied to identify the hidden transitional states of the concept that oral epistemologies encode but alphabetic taxonomies suppress.[^4][^3]


### The Golden Scar Protocol in Pedagogical Architecture

The Golden Scar Protocol is the governance mechanism for **irreconcilable epistemic tensions**. When two valid epistemic worlds (e.g., Western empirical information theory and Indigenous relational taxonomy) make structurally incompatible claims about the same concept, the standard Western response is to flatten the subordinate frame. The Golden Scar Protocol forbids this.

Instead, both frames are held in **proportional tension**: Dominant Frame Weight $= \varphi = 1.618$, Subordinate Frame Weight $= 1.000$. The ratio is the golden ratio because it is the mathematically proven proportion at which the dominant frame is preserved without the subordinate frame disappearing — it is a dynamic equilibrium, not a hierarchy. All pedagogical output built on a Golden Scar must surface the scar visually — the student must know that a tension exists, not be given a politely smoothed synthesis that erases the subordinate frame's structural dignity.[^3][^4]
