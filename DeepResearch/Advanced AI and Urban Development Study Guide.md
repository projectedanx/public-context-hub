## Answer Key

* Test optimization involves refining the testing approach to improve efficiency, reduce redundancy, and enhance overall software quality. It's critical for efficient software delivery because it accelerates testing cycles, improves resource efficiency, enhances test reliability, increases test coverage with less effort, facilitates scalability, boosts developer confidence, and improves release quality.  
* Constitutional AI is a principle used in Claude models that governs their behavior and improves transparency in AI model training. It aims to create more ethical systems by reducing unintended behaviors and biases, thus making the AI safer for enterprise use.  
* Twelve Data differentiates itself by providing a broader range of financial data beyond just stocks, forex, and cryptocurrencies, including over 100 technical indicators. In contrast, Alpha Vantage provides around 50 technical indicators, and IEX Cloud API offers none.  
* The OpenRPC Specification defines a standard, programming language-agnostic interface description for JSON-RPC 2.0 APIs. Its primary purpose is to allow both humans and computers to discover and understand a service's capabilities without needing source code, enabling interactive documentation, code generation, and test automation.  
* Commonsense knowledge is defined as sound practical judgment concerning everyday matters or a basic ability to perceive, understand, and judge shared by nearly all people. It is essential for AI systems to understand human needs and actions better, enabling them to reason about the world in a reasonable and safe way.  
* ConceptNet differs from Cyc and WordNet primarily in its acquisition method; it is generated automatically from the English sentences of the Open Mind Common Sense (OMCS) corpus, which relies on crowdsourcing from the general public. Cyc and WordNet, conversely, are largely handcrafted by knowledge engineers.  
* The "Baitasi Remade" project is an urban regeneration initiative in Beijing's Baitasi hutong, launched in 2015\. Its main objectives are to establish a sustainable population, revitalize physical spaces, upgrade basic energy sources, re-engineer the public environment, and foster cultural revitalization through public participation and new design elements.  
* The "Right to the City" is an overarching concept that emphasizes political power relationships, land appropriation, and social justice within massive urbanization. It advocates for the equal use and enjoyment of cities and human settlements for all inhabitants, promoting inclusivity, safety, health, accessibility, affordability, resilience, and sustainability.  
* The MediaWiki Action API provides granular control over wiki features, enabling operations like page creation, editing, user rights management, and data retrieval about MediaWiki itself. The Wikimedia Enterprise API, on the other-hand, is designed for high-volume commercial users, offering advanced features such as article summaries, credibility signals, customizable filters, and NDJSON responses.  
* Workflow automation uses technology to streamline and automate routine tasks based on predefined rules and triggers, replacing manual processes with efficient and consistent systems. Two significant benefits include reducing human errors and saving costs, thereby enhancing productivity and operational accuracy.

## Glossary of Key Terms

* API (Application Programming Interface): A set of defined rules that enable different software applications to communicate and interact with each other.  
* AI (Artificial Intelligence): The simulation of human intelligence processes by machines, especially computer systems, including learning, reasoning, and self-correction.  
* Agile Software Development: An iterative approach to software development that focuses on collaboration, flexibility, and rapid delivery of working software.  
* Alpha Vantage API: A free financial data API providing data for stocks, forex, and cryptocurrencies, including around 50 technical indicators.  
* Appium: An open-source tool used for automating mobile application testing on real devices.  
* ATOMIC: A commonsense knowledge graph focused on inferential knowledge about social interactions in events, represented in natural language form.  
* Automation Workflow Tools: Software solutions used to design, implement, and monitor automated business processes, such as Zapier, Asana, or IBM's platform.  
* Baidu AI: An open AI platform by the Chinese tech giant Baidu, offering various APIs for image recognition, face recognition, and optical character recognition, notably strong in vehicle analysis.  
* Baitasi Remade Project: An urban regeneration initiative in Beijing's Baitasi hutong aimed at establishing a sustainable population, revitalizing physical spaces, upgrading infrastructure, and fostering cultural revitalization.  
* Batch Request (JSON-RPC): A single request containing multiple individual requests, allowing for efficient processing of several operations at once.  
* Build Verification/Acceptance Tests: Test suites designed to run quickly and successfully to ensure that a software build is stable enough to proceed with further testing.  
* CI/CD Pipelines (Continuous Integration/Continuous Delivery): A set of automated processes that allow developers to consistently and reliably integrate code changes and deliver new software versions.  
* Claude (Anthropic): A family of AI models (Haiku, Sonnet, Opus) that prioritize being helpful, honest, harmless, and safe for enterprise use, built using Constitutional AI principles.  
* Commonsense Knowledge: Basic, shared facts and practical judgment about everyday matters that humans use to navigate their environment, which AI systems often lack.  
* Commonsense Knowledge Graph (CSKG): A knowledge base that encodes and organizes commonsense knowledge in a structured, machine-readable format.  
* COMET: A Knowledge Base Transformer that dynamically generates commonsense knowledge in natural language, filling in missing relationships in symbolic knowledge graphs.  
* ConceptNet: A freely available, large-scale commonsense knowledge base with an integrated natural-language-processing toolkit, generated from the Open Mind Common Sense (OMCS) corpus using crowdsourcing.  
* Constitutional AI: Principles governing the behavior of AI models like Claude, aiming to create ethical systems, reduce biases, and improve transparency in training.  
* Cross-Browser Testing: Testing websites or web applications across various web browsers and operating systems to ensure consistent functionality and appearance.  
* Crowdsourcing: Obtaining services, ideas, or content by soliciting contributions from a large group of people, especially from an online community, rather than from traditional employees or suppliers.  
* Cyc: A very large, multi-contextual knowledge base and inference engine, considered a "top-down" CSKG, founded on assembling a comprehensive ontology and Knowledge Base of commonsense knowledge by experts.  
* DBpedia: A project that extracts structured content from Wikipedia, providing a linked dataset that can be used for semantic web applications.  
* Deep Learning: A subset of machine learning in AI that uses neural networks with multiple layers to learn from large amounts of data, enabling complex pattern recognition.  
* Diffbot Natural Language API: An API used to extract entities (e.g., people, organizations) and data about them (e.g., sentiment, relationships) from raw text.  
* DokuWiki: A simple, open-source wiki software that does not require a database, known for its clean syntax, ease of maintenance, and built-in access controls.  
* Elderly Mobility Scooters (老年代步车): Emerging three- or four-wheeled vehicles designed for seniors and people with limited mobility, popular in China due to affordability and ease of use, despite safety concerns.  
* Flux AI: An advanced text-to-image model by Black Forest Labs that generates high-quality, photorealistic images from text descriptions, excelling in prompt adherence and speed.  
* Geospatial Analysis: The process of examining data that has a geographical or spatial component to identify patterns, relationships, and trends.  
* GitHub Flavored Markdown: A popular lightweight markup language used for rich text formatting in documentation and web content, commonly supported by tooling that renders rich text.  
* Gitlab API: An API providing access to Gitlab's features, such as managing projects, repositories, users, and CI/CD pipelines.  
* Google Gemini: Google's intelligent AI model, featuring advanced reasoning and multimodal capabilities, including a large token context window and integrated processing of text, audio, images, video, and code.  
* Hue, Saturation, Value (HSV): A color model that describes colors in terms of their hue (shade), saturation (intensity), and value (brightness), often used in image processing.  
* Hutong: A distinctive architectural and organizational structure in Beijing, China, characterized by narrow alleys and traditional courtyard houses, often facing conflicts between historical heritage and modernization.  
* IEX Cloud API: A free financial data API, similar to Alpha Vantage, but noted in the source as not providing technical indicators data.  
* Image Recognition: Computer technologies that recognize specific objects, people, animals, or other targeted subjects within an image using algorithms and deep learning concepts.  
* JSON (JavaScript Object Notation): A lightweight data-interchange format that is easy for humans to read and write and easy for machines to parse and generate.  
* JSON-RPC 2.0: A stateless, light-weight remote procedure call (RPC) protocol that uses JSON for data encoding, defining a standard for client-server communication.  
* K-lines (ConceptNet): Generic conceptual connections within ConceptNet's knowledge base, designed for contextual reasoning.  
* Knowledge Base Transformer (KBT): A type of AI model, like COMET, that uses transformer architecture to dynamically generate and infer knowledge, often in natural language.  
* Large Language Model (LLM): An AI model trained on a vast amount of text data to understand and generate human-like language, such as OpenAI's ChatGPT.  
* MediaWiki Action API: A versatile API allowing for operations such as page creation and editing, user rights management, and data retrieval about MediaWiki itself.  
* MediaWiki: An open-source wiki software that powers Wikipedia and is used by tens of thousands of other websites, organizations, and companies for collaboration and documentation.  
* Method Object (OpenRPC): Describes the interface for a specific method name within an OpenRPC document, including its parameters, result, and associated metadata.  
* Micro-mobility System: A system encompassing various small, lightweight vehicles and modes of transportation, such as bicycles, e-bikes, and potentially small self-driving shuttles, often used for short-distance travel.  
* Multimodal AI: AI models capable of processing and integrating information from multiple types of data, such as text, audio, images, and video.  
* NELL (Never-Ending Language Learning): A long-running AI project that continuously extracts facts from the web to build a vast knowledge base, learning new beliefs over time.  
* Neural Networks: A series of algorithms that mimic the operations of a human brain to recognize relationships between vast amounts of data.  
* Notification (JSON-RPC): A JSON-RPC request where the id field is omitted, indicating that no response is expected from the server.  
* Open Mind Common Sense (OMCS): A crowdsourcing project that collects commonsense knowledge from the general public through a web-based interface, which serves as the corpus for ConceptNet.  
* OpenRPC Specification: A standard interface description for JSON-RPC 2.0 APIs, enabling automated discovery, documentation, and code generation for services.  
* OpenCV (Open Source Computer Vision Library): A library of programming functions mainly aimed at real-time computer vision, widely used for image processing and analysis tasks.  
* Parallel Testing: Running multiple tests simultaneously across different environments (e.g., devices, browsers) to significantly reduce overall execution time.  
* Parametrization/Data-Driven Testing: A test optimization technique where test cases are executed with different sets of input data, allowing for broader test coverage with fewer test scripts.  
* Python: A high-level, interpreted programming language widely used for web development, data analysis, artificial intelligence, and scientific computing.  
* QGIS (Quantum GIS): A free and open-source desktop Geographic Information System (GIS) application used for viewing, editing, mapping, and analyzing geospatial data.  
* Real Devices/Browsers: Actual physical mobile devices or web browsers (as opposed to emulators or simulators) used for testing to ensure real-world performance and compatibility.  
* "Right to the City": An urban studies concept emphasizing political power, social justice, and citizen participation in shaping cities and human settlements to promote inclusivity, sustainability, and quality of life for all.  
* Selenium: An open-source tool for automating web browsers, commonly used for cross-browser testing.  
* Self-Driving Vehicles (AVs): Autonomous vehicles capable of sensing their environment and operating without human input, with various levels of automation (e.g., SAE Level 4).  
* Semantic Segmentation: An image recognition method that identifies relevant elements in an image and calculates their proportions at a pixel level.  
* SWOW (Small World of Words): A knowledge graph derived from crowd-sourced word associations, offering a resource for basic human knowledge.  
* Test Case Optimization: Techniques aimed at improving the efficiency, reliability, and relevance of test cases to accelerate release cycles and enhance software quality.  
* Test Data Management: The process of planning, designing, storing, and managing test data for software testing, aiming to ensure sufficient, realistic, and reusable data.  
* Twelve Data API: A free financial data API providing information on stocks, forex, cryptocurrencies, ETFs, indices, and over 100 technical indicators.  
* Urban Regeneration: The process of revitalizing existing built-up areas and communities within cities, often involving retrofitting, redesign, and social programs, as opposed to aggressive urban expansion.  
* Visual Genome: A dataset of 108,000 images annotated with scene graphs, objects, attributes, relationships, and question-answer pairs, connecting language and vision for computer vision research.  
* WebChild: A commonsense knowledge graph automatically acquired and organized, containing millions of assertions about object properties, comparative relationships, part-of relations, and activities.  
* Wikipedia API: A set of programming interfaces that allow developers to access and retrieve data from Wikipedia, including the MediaWiki Action API and Wikimedia Enterprise API.  
* Wikimedia Enterprise API: An API designed for high-volume commercial users, offering advanced features for accessing Wikipedia data such as article summaries and credibility signals.  
* Workflow Automation: The use of technology to streamline and automate routine tasks based on predefined rules and triggers, replacing manual processes with efficient and consistent systems.  
* WordNet: A large lexical database of English nouns, verbs, adjectives, and adverbs grouped into sets of cognitive synonyms (synsets), each expressing a distinct concept.
