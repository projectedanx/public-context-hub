The capsule is defined as much by what it **omits** as what it contains. The `compression_ratio` and `redactions` fields make the absence *explicit*. Visitors know there's a deeper report. The membrane is transparent about its own boundaries.


## Orchestration Pseudocode

The runtime that drives the council:

```python
async def run_council(raw_context: dict, config: CouncilConfig) -> FinalManifest:
    envelope = build_envelope(raw_context, config, round="discovery")

    # Round 1: Discovery (parallel)
    briefs = await asyncio.gather(
        call_seat("strategist", envelope),
        call_seat("immunologist", envelope),
        call_seat("linguist", envelope),
        call_seat("engineer", envelope),
        call_seat("historian", envelope),
    )

    # Confidence gate
    low_confidence_count = sum(
        1 for b in briefs if b["discovery_brief"]["confidence"] < 0.5
    )
    if low_confidence_count >= 3:
        return flag_for_human_review(briefs)

    # Round 2: Synthesis (conductor)
    envelope_r2 = build_envelope(
        raw_context, config,
        round="synthesis",
        briefs=briefs,
    )
    draft = await call_seat("conductor", envelope_r2)

    # Round 3: Critique (parallel)
    envelope_r3 = build_envelope(
        raw_context, config,
        round="critique",
        draft_manifest=draft["sovereignAgentManifest"],
    )
    critiques = await asyncio.gather(
        call_seat("strategist", envelope_r3),
        call_seat("immunologist", envelope_r3),
        call_seat("linguist", envelope_r3),
        call_seat("engineer", envelope_r3),
        call_seat("historian", envelope_r3),
    )

    # Short-circuit if unanimous approval
    if all(c["verdict"] == "approve" for c in critiques):
        return mint_manifest(draft, critiques, shortcircuit=True)

    # Round 4: Finalization (conductor)
    change_requests = [
        c for c in critiques if c["verdict"] == "change_request"
    ]
    envelope_r4 = build_envelope(
        raw_context, config,
        round="finalization",
        draft_manifest=draft["sovereignAgentManifest"],
        change_requests=change_requests,
    )
    final = await call_seat("conductor", envelope_r4)

    return sign_and_store(final)
```


```text

```

Each persona reads upstream artifacts but doesn't **sign** its output or reference which specific upstream claims it relied on. When the Tester reports a failure, you can't trace whether the fault was in the Spec, the Architecture, the Plan, or the Implementation.

Now every artifact has a **birth certificate**. When P6 finds a bug, you can trace the causal chain backward through content hashes.

Every other persona outputs a single markdown file. The Implementer outputs **arbitrary source code** — the attack surface explodes. There's no:
- File count limit
- Allowed file extension list
- Directory boundary enforcement
- Token budget ceiling

The Router checks **existence**, not **quality**. If P2 produces a malformed spec, the Router sees `docs/spec.md exists` and happily routes to P3. Garbage flows downstream.

This doesn't need an LLM — it's structural validation. Does the spec have all required sections? Are FRs numbered? Are priorities assigned? Template compliance is checkable with regex or a lightweight parser.

You're building a system with Token Allocation and Budget Management in SCOS, but the pipeline has no cost tracking. Each persona should report its token consumption, and the Router should enforce a session budget:
