$$
IM(A,B) = \begin{pmatrix}
\dim(I^A \cap I^B) & \dim(I^A \cap \partial B) & \dim(I^A \cap E^B) \\
\dim(\partial A \cap I^B) & \dim(\partial A \cap \partial B) & \dim(\partial A \cap E^B) \\
\dim(E^A \cap I^B) & \dim(E^A \cap \partial B) & \dim(E^A \cap E^B)
\end{pmatrix}
$$

For a **Signed Distance Field** $d_A : \mathbb{R}^3 \to \mathbb{R}$ representing solid body $A$, the three topological regions map exactly to the three sign classes of the scalar output:[^3][^4]


| **DE-9IM Region** | **SDF Codomain** | **Physical Meaning** |

| Exterior $E^A$ | $d_A(\mathbf{p}) > 0$ | Point $\mathbf{p}$ is outside body $A$ |

The necessary and sufficient condition for this isomorphism to hold without approximation is the **Eikonal property**: $|\nabla d_A(\mathbf{p})| = 1$ almost everywhere [^4][^5]. ### 2.1 The Boundary Sampling Protocol

4. **Interior probe**: Select a guaranteed interior point of each body (e.g., $\mathbf{p}_{int}^B$ where $d_B(\mathbf{p}) \ll 0$), evaluate against the other SDF to populate the $I^A \cap I^B$ cell.

    STEP_OVERSHOOT      = "ALGORITHMIC_SHAME_STEP_OVERSHOOT_FALSIFICATION"

# ─────────────────────────────────────────────────────────────────────────────
# SDF PRIMITIVE DEFINITIONS (Exact Analytic, Lipschitz=1)
# ─────────────────────────────────────────────────────────────────────────────

class SDFBody(BaseModel):
    """
    A closed-form SDF definition. The callable sdf_fn must satisfy |∇d|=1.
    gradient_fn provides ∇d analytically (preferred) or via finite differences.
    """
    body_id: str
    nominal_radius_m: float = Field(..., description="Nominal dimension in meters")
    sdf_fn: Callable[[np.ndarray], float] = Field(
        ..., description="Exact SDF: R^3 -> R, satisfying Eikonal property"
    )
    gradient_fn: Callable[[np.ndarray], np.ndarray] = Field(
        ..., description="∇d: R^3 -> R^3, must have magnitude 1 at non-degenerate points"
    )

    class Config:
        arbitrary_types_allowed = True

    def evaluate(self, p: np.ndarray) -> float:
        return float(self.sdf_fn(p))

    def gradient(self, p: np.ndarray) -> np.ndarray:
        return self.gradient_fn(p)

    def validate_eikonal_at(self, p: np.ndarray, tol: float = 0.01) -> bool:
        """Verify |∇d(p)| ≈ 1. Returns False on Eikonal violation."""
        grad_mag = float(np.linalg.norm(self.gradient(p)))
        return abs(grad_mag - 1.0) < tol

# ─────────────────────────────────────────────────────────────────────────────
# ISO 286-1 TOLERANCE ORACLE
# ─────────────────────────────────────────────────────────────────────────────

class ISOToleranceBand(BaseModel):
    """
    ISO 286-1:2010 tolerance band for a given fit designation and nominal size.
    All values in meters.
    """
    fit_designation: str          # e.g. "H7/g6"
    nominal_diameter_m: float
    bore_upper_dev_m: float       # ES (upper deviation of hole)
    bore_lower_dev_m: float       # EI (lower deviation of hole)
    shaft_upper_dev_m: float      # es (upper deviation of shaft)
    shaft_lower_dev_m: float      # ei (lower deviation of shaft)

    @property
    def epsilon_iso(self) -> float:
        """
        The functional epsilon: half the sum of both tolerance grades.
        This is the minimum detectable fit change that matters mechanically.
        """
        bore_band  = abs(self.bore_upper_dev_m  - self.bore_lower_dev_m)
        shaft_band = abs(self.shaft_upper_dev_m - self.shaft_lower_dev_m)
        return (bore_band + shaft_band) / 2.0

    @property
    def max_clearance(self) -> float:
        """ES - ei: maximum possible clearance (negative = always interference)."""
        return self.bore_upper_dev_m - self.shaft_lower_dev_m

    @property
    def min_clearance(self) -> float:
        """EI - es: minimum clearance (negative = minimum interference)."""
        return self.bore_lower_dev_m - self.shaft_upper_dev_m

    @property
    def nominal_iso_fit_class(self) -> ISOFitClass:
        if self.min_clearance > 0:
            return ISOFitClass.CLEARANCE
        elif self.max_clearance < 0:
            return ISOFitClass.INTERFERENCE
        else:
            return ISOFitClass.TRANSITION

# ─────────────────────────────────────────────────────────────────────────────
# DE-9IM MATRIX MODEL
# ─────────────────────────────────────────────────────────────────────────────

class DE9IMMatrix(BaseModel):
    """
    Represents the 3x3 DE-9IM intersection matrix as a 9-character string.
    Row order: [Interior, Boundary, Exterior] of body A
    Col order: [Interior, Boundary, Exterior] of body B
    """
    matrix_string: str = Field(..., min_length=9, max_length=9)

    @field_validator("matrix_string")
    @classmethod
    def validate_cells(cls, v: str) -> str:
        valid = set("F012T")
        assert all(c in valid for c in v), f"Invalid DE-9IM cell value in: {v}"
        return v

    @property
    def II(self) -> str: return self.matrix_string[^0]  # Interior(A) ∩ Interior(B)
    @property
    def IB(self) -> str: return self.matrix_string[^1]  # Interior(A) ∩ Boundary(B)
    @property
    def IE(self) -> str: return self.matrix_string[^2]  # Interior(A) ∩ Exterior(B)
    @property
    def BI(self) -> str: return self.matrix_string[^3]  # Boundary(A) ∩ Interior(B)
    @property
    def BB(self) -> str: return self.matrix_string[^4]  # Boundary(A) ∩ Boundary(B)
    @property
    def BE(self) -> str: return self.matrix_string[^5]  # Boundary(A) ∩ Exterior(B)
    @property
    def EI(self) -> str: return self.matrix_string[^6]  # Exterior(A) ∩ Interior(B)
    @property
    def EB(self) -> str: return self.matrix_string[^7]  # Exterior(A) ∩ Boundary(B)
    @property
    def EE(self) -> str: return self.matrix_string[^8]  # Exterior(A) ∩ Exterior(B)

    def to_iso_fit_class(self) -> ISOFitClass:
        """
        Map DE-9IM string to ISO fit class.
        This is the core algebraic translation function.
        """
        II = self.II
        BB = self.BB

        if II == "F" and BB == "F":
            # Disjoint — clearance fit, positive gap
            return ISOFitClass.CLEARANCE
        elif II == "F" and BB in ("0", "1"):
            # Touches — boundary contact only, transition fit
            return ISOFitClass.TRANSITION
        elif II in ("2", "3", "T"):
            # Overlaps / Contains — interiors intersect, interference fit
            return ISOFitClass.INTERFERENCE
        else:
            # Paraconsistent zone: ambiguous (should not occur with valid SDFs)
            return ISOFitClass.PARACONSISTENT

# ─────────────────────────────────────────────────────────────────────────────
# TOLERANCE AGENT — CORE ENGINE
# ─────────────────────────────────────────────────────────────────────────────

class ToleranceAgent:
    """
    SCOS DE-9IM Tolerance Agent.
    Inputs:  Two SDFBody objects + ISOToleranceBand
    Output:  AssemblyFitResult (JSON-LD Knowledge Capsule)

    Operates entirely in continuous SDF space.
    Mesh, voxel, or polygon data is NEVER ingested.
    """

    EIKONAL_TOLERANCE    = 0.01   # Allowable deviation of |∇d| from 1.0
    K_SAFETY             = 4.0    # Para-band safety factor vs epsilon_ISO
    N_BOUNDARY_SAMPLES   = 512    # Boundary sampling resolution
    N_REFINE_SAMPLES     = 64     # Paraconsistent zone refinement samples
    MAX_RAYMARCH_STEPS   = 1024   # Hard cap on raymarching iterations

    def __init__(self, body_a: SDFBody, body_b: SDFBody, iso_band: ISOToleranceBand):
        self.A    = body_a
        self.B    = body_b
        self.iso  = iso_band
        self.epsilon_ray  = iso_band.epsilon_iso / 2.0
        self.epsilon_para = iso_band.epsilon_iso / self.K_SAFETY

    # ── RAYMARCHING: Converge to zero-level set of target SDF ──────────────

    def _raymarch_to_surface(
        self,
        sdf_target: SDFBody,
        ray_origin: np.ndarray,
        ray_dir: np.ndarray
    ) -> Optional[np.ndarray]:
        """
        Sphere-trace along ray_dir from ray_origin until sdf_target ≈ 0.
        Returns surface point or None if no intersection found.
        Enforces step_size <= epsilon_ray to prevent Resolution Collapse.
        """
        p = ray_origin.copy().astype(np.float64)
        for _ in range(self.MAX_RAYMARCH_STEPS):
            d = sdf_target.evaluate(p)
            if abs(d) < self.epsilon_ray:
                return p
            # Enforce minimum step to prevent infinite loop at saddle points
            step = max(abs(d), self.epsilon_ray * 0.1)
            # Cap step at epsilon_ray to enforce Resolution Collapse guarantee
            step = min(step, self.epsilon_ray * 2.0)
            if d < 0:
                step = -step  # Inside body: march toward surface
            p = p + ray_dir * step
        return None  # No convergence within step budget

    # ── EIKONAL VALIDATION ─────────────────────────────────────────────────

    def _validate_eikonal(self, body: SDFBody, sample_points: list[np.ndarray]) -> bool:
        """
        Verify |∇d| ≈ 1 at a set of boundary sample points.
        Returns False (triggers EpistemicEscrow halt) if any point fails.
        """
        for p in sample_points:
            if not body.validate_eikonal_at(p, self.EIKONAL_TOLERANCE):
                return False
        return True

    # ── BOUNDARY SAMPLING PASS ─────────────────────────────────────────────

    def _sample_boundary(self, body: SDFBody) -> list[np.ndarray]:
        """
        Generate boundary points on body by ray-casting from a sphere
        of radius 3x nominal_radius, uniformly distributed via Fibonacci lattice.
        """
        boundary_points = []
        r_sample = body.nominal_radius_m * 3.0
        # Fibonacci lattice on sphere for uniform angular coverage
        golden = (1 + math.sqrt(5)) / 2
        for i in range(self.N_BOUNDARY_SAMPLES):
            theta = 2 * math.pi * i / golden
            phi   = math.acos(1 - 2 * (i + 0.5) / self.N_BOUNDARY_SAMPLES)
            origin = np.array([
                r_sample * math.sin(phi) * math.cos(theta),
                r_sample * math.sin(phi) * math.sin(theta),
                r_sample * math.cos(phi)
            ])
            direction = -origin / np.linalg.norm(origin)  # Inward ray
            hit = self._raymarch_to_surface(body, origin, direction)
            if hit is not None:
                boundary_points.append(hit)
        return boundary_points

    # ── DE-9IM MATRIX CONSTRUCTION ─────────────────────────────────────────

    def _build_de9im_matrix(
        self,
        boundary_A: list[np.ndarray],
        boundary_B: list[np.ndarray],
        d_min: float
    ) -> DE9IMMatrix:
        """
        Classify boundary samples to build the 9-intersection string.
        Uses exact SDF evaluations — no polygon arithmetic.
        """
        ep = self.epsilon_para

        # ── Cell [0,0]: I(A) ∩ I(B) ──
        # Check if any point deep inside A is also inside B
        # Use A's centroid as interior probe
        interior_A = np.array([0.0, 0.0, 0.0])  # Assumes A centered at origin
        II_val = "2" if self.B.evaluate(interior_A) < -ep else "F"

        # ── Cell [1,1]: ∂A ∩ ∂B ──
        bb_count = sum(1 for p in boundary_A if abs(self.B.evaluate(p)) < ep)
        BB_val = "0" if bb_count > 0 else "F"

        # ── Cell [1,0]: ∂A ∩ I(B) ──
        bi_count = sum(1 for p in boundary_A if self.B.evaluate(p) < -ep)
        BI_val = "2" if bi_count > 0 else "F"

        # ── Cell [0,1]: I(A) ∩ ∂B ──
        ib_count = sum(1 for p in boundary_B if self.A.evaluate(p) < -ep)
        IB_val = "2" if ib_count > 0 else "F"

        # Exterior cells — always 2 for bounded 3D solids
        matrix_str = f"{II_val}{IB_val}2{BI_val}{BB_val}2212"
        return DE9IMMatrix(matrix_string=matrix_str)

    # ── PARACONSISTENT RESOLUTION ──────────────────────────────────────────

    def _paraconsistent_resolve(self, p_center: np.ndarray) -> float:
        """
        Sample N_REFINE_SAMPLES points on a sphere of radius epsilon_para/2
        centered at p_center. Return fraction with delta < 0 (interference side).
        """
        ep_r = self.epsilon_para / 2.0
        count_neg = 0
        for i in range(self.N_REFINE_SAMPLES):
            angle = 2 * math.pi * i / self.N_REFINE_SAMPLES
            offset = np.array([ep_r * math.cos(angle), ep_r * math.sin(angle), 0.0])
            p_probe = p_center + offset
            delta = self.A.evaluate(p_probe) - self.B.evaluate(p_probe)
            if delta < 0:
                count_neg += 1
        return count_neg / self.N_REFINE_SAMPLES

    # ── TOPOLOGICAL DERIVATIVE FIT FORCE ESTIMATE ─────────────────────────

    def _estimate_press_force(
        self,
        d_min: float,
        iso_band: ISOToleranceBand,
        youngs_modulus_pa: float = 200e9,  # Default: steel
        poissons_ratio: float = 0.3
    ) -> float:
        """
        First-order Lamé press-fit force estimate from SDF interference depth.
        δ = |d_min|, r = nominal radius.
        """
        if d_min >= 0:
            return 0.0
        delta     = abs(d_min)  # Interference depth in meters
        r         = iso_band.nominal_diameter_m / 2.0
        # Simplified thin-ring Lamé formula
        P_contact = (delta * youngs_modulus_pa) / (2.0 * r)  # Contact pressure [Pa]
        # Force = pressure × contact area (assume axial contact length = r)
        F_press   = P_contact * (2 * math.pi * r * r)
        return F_press

    # ── MAIN EXECUTE ──────────────────────────────────────────────────────

    def execute(
        self,
        assembly_id: str,
        thermal_delta_T_K: Optional[float] = None,
        cte_per_K: float = 12e-6,        # Default: steel CTE
        scos_timestamp_ns: int = 0
    ) -> AssemblyFitResult:
        """
        Primary Tolerance Agent execution pipeline.
        Returns a fully populated AssemblyFitResult JSON-LD capsule.
        """
        shame_code = EpistemicShameCode.NONE

        # ── STEP 1: Apply thermal perturbation if present ──────────────────
        thermal_applied = False
        if thermal_delta_T_K is not None:
            thermal_shift = cte_per_K * thermal_delta_T_K * self.A.nominal_radius_m
            original_sdf_fn = self.A.sdf_fn
            self.A = SDFBody(
                body_id=self.A.body_id + "_thermal",
                nominal_radius_m=self.A.nominal_radius_m * (1 + cte_per_K * thermal_delta_T_K),
                sdf_fn=lambda p, shift=thermal_shift, fn=original_sdf_fn: fn(p) - shift,
                gradient_fn=self.A.gradient_fn
            )
            thermal_applied = True

        # ── STEP 2: Sample boundaries ──────────────────────────────────────
        boundary_A = self._sample_boundary(self.A)
        boundary_B = self._sample_boundary(self.B)

        # ── STEP 3: Validate Eikonal (EpistemicEscrow gate) ───────────────
        eikonal_ok = (
            self._validate_eikonal(self.A, boundary_A[:16]) and
            self._validate_eikonal(self.B, boundary_B[:16])
        )
        if not eikonal_ok:
            shame_code = EpistemicShameCode.EIKONAL_VIOLATION
            return AssemblyFitResult(
                assembly_id=assembly_id,
                body_a_id=self.A.body_id,
                body_b_id=self.B.body_id,
                iso_tolerance_band=self.iso.fit_designation,
                de9im_string="FFFFFFFFF",
                iso_fit_class=ISOFitClass.EIKONAL_VIOLATION,
                d_min_m=float("nan"),
                epsilon_iso_m=self.iso.epsilon_iso,
                epsilon_para_m=self.epsilon_para,
                eikonal_valid=False,
                epistemic_shame_code=shame_code,
                thermal_expansion_applied=thermal_applied,
                thermal_delta_T_K=thermal_delta_T_K,
                confidence=0.0,
                scos_timestamp_ns=scos_timestamp_ns
            )

        # ── STEP 4: Compute d_min (minimum signed distance between surfaces)
        d_values_A_on_B = [self.A.evaluate(p) for p in boundary_B]
        d_values_B_on_A = [self.B.evaluate(p) for p in boundary_A]
        d_min = min(min(d_values_A_on_B), min(d_values_B_on_A))

        # ── STEP 5: Check for paraconsistent zone ─────────────────────────
        confidence = 1.0
        if abs(d_min) < self.epsilon_para:
            shame_code = (
                EpistemicShameCode.THERMAL_PERTURBATION
                if thermal_applied
                else EpistemicShameCode.BOUNDARY_SUPERPOSITION
            )
            # Find the closest boundary point for refinement
            idx_min = np.argmin(np.abs(d_values_B_on_A))
            p_critical = boundary_A[idx_min]
            f_neg = self._paraconsistent_resolve(p_critical)
            confidence = 1.0 - (abs(d_min) / self.epsilon_para)

        # ── STEP 6: Build DE-9IM matrix ───────────────────────────────────
        de9im = self._build_de9im_matrix(boundary_A, boundary_B, d_min)

        # ── STEP 7: Classify fit ──────────────────────────────────────────
        if shame_code in (
            EpistemicShameCode.BOUNDARY_SUPERPOSITION,
            EpistemicShameCode.THERMAL_PERTURBATION
        ):
            iso_fit = ISOFitClass.PARACONSISTENT
        else:
            iso_fit = de9im.to_iso_fit_class()

        # ── STEP 8: Press-fit force estimate (interference only) ──────────
        press_force = None
        if iso_fit == ISOFitClass.INTERFERENCE:
            press_force = self._estimate_press_force(d_min, self.iso)

        return AssemblyFitResult(
            assembly_id=assembly_id,
            body_a_id=self.A.body_id,
            body_b_id=self.B.body_id,
            iso_tolerance_band=self.iso.fit_designation,
            de9im_string=de9im.matrix_string,
            iso_fit_class=iso_fit,
            d_min_m=d_min,
            epsilon_iso_m=self.iso.epsilon_iso,
            epsilon_para_m=self.epsilon_para,
            eikonal_valid=eikonal_ok,
            epistemic_shame_code=shame_code,
            thermal_expansion_applied=thermal_applied,
            thermal_delta_T_K=thermal_delta_T_K,
            press_fit_force_N_estimate=press_force,
            confidence=confidence,
            scos_timestamp_ns=scos_timestamp_ns
        )

## Part VII — Five Lenses: Knowledge Through Perspective

### Lens 1 — Critical Spatial Lens: Power Dynamics of the Gap

The topological boundary in a mechanical assembly is not a passive geometric line — it is a **locus of force concentration**. In an interference fit, the bore is the yielding component: its hoop stress $\sigma_\theta = P_{contact} \cdot r / t$ (where $t$ is wall thickness) determines whether it expands elastically or yields plastically. The shaft is typically in compression, the bore in tension. The DE-9IM `212101212` string — indicating $I^A \cap I^B \neq \emptyset$ — is topologically encoding this power asymmetry: the shaft's volume has colonized the bore's volume, and the bore must yield. The SDF $d_{min} < 0$ value is the magnitude of this territorial transgression. The Tolerance Agent, by reporting $d_{min}$ alongside the fit class, provides the immediate input to compute which component bears the elastic penalty.[^12][^16][^9]

### Lens 2 — Metabolic Lens: Energy Topology

Interference fits are mechanical batteries. The press-fit force performs work $W = F_{press} \times \delta$ (where $\delta = |d_{min}|$), stored as elastic strain energy in the hoop-stressed bore [^16]. This energy is **topologically locked** — it cannot dissipate until the DE-9IM string transitions from `212101212` back toward `FF2FF1212` (by thermal relaxation, vibration fatigue, or press-out force). The SCOS metabolic model maps this as a high-impedance energy bottleneck at the topological boundary. Clearance fits, by contrast, represent **metabolic openness**: the `FF2FF1212` gap topology is a lubrication reservoir and thermal expansion buffer — the absence is not a void but a functional physiological clearance, like the synovial gap in a joint. The Tolerance Agent's clearance void (Lens 5) is metabolically necessary [^10].

### Lens 3 — Ludic Lens: Rules of the Boundary Game

The tolerance game has three legal moves: the shaft can be undersized (clearance wins), oversized (interference wins), or exactly nominal (transition — a draw). The rules are enforced by the zero-level set $d = 0$. The "cheat" condition — where the continuous physics engine produces an undetected penetration — is formally the **Resolution Collapse** event: the raymarching step size exceeds the gap width, and the ray "teleports" through the boundary. The Tolerance Agent prevents this cheat by the step-size bound $\epsilon_{ray} \leq \epsilon_{ISO}/2$. If this bound is ever violated (e.g., adaptive step size optimization naively maximizes step size near surfaces), the Falsification Condition of this DRP is triggered and the SCOS EpistemicEscrow halts execution. The game has exactly one rule: **the step must never exceed half the tolerance**.[^15][^3]

### Lens 4 — Posthumanist / New Materialist Lens: Agency of Real Surfaces

The SDF represents a **Platonic ideal** — a mathematical surface of zero thickness, perfectly smooth, exactly described by an analytic formula. Real manufactured surfaces possess **micro-topological agency**: surface roughness $R_a$, waviness, and form error all modify the effective contact boundary. A shaft ground to ISO tolerance H7/g6 has a surface roughness on the order of $R_a = 0.8$ μm (Ra for ground shaft), while the tolerance gap is ~7 μm — meaning the roughness amplitude is approximately 10% of the functional tolerance. The SDF does not represent this micro-agency. The **Non-Euclidean Wear Hypothesis** partially addresses this: by treating $R_a$ as the initial amplitude of the wear-field perturbation $\dot{d}$, the agent can initialize a stochastic surface roughness field atop the ideal SDF and propagate it through the Archard wear law. This bridges the Platonic SDF to material reality without abandoning the continuous framework.[^20][^21][^18]

### Lens 5 — Lens of Silence and Absence: The Functional Clearance Void

The clearance gap in a `FF2FF1212` Disjoint topology is not empty space. It is an **active functional absence** with engineering responsibilities: it carries lubricant film (hydrodynamic bearing theory: film thickness $h \propto \mu v / P$, where $\mu$ is viscosity, $v$ velocity, $P$ pressure), dissipates heat from frictional contact, accommodates thermal expansion differentials, and permits kinematic motion. If the SDF $d_{min}$ of a sliding fit decreases below the minimum oil film thickness $h_{min}$ (typically 1-3 μm for precision bearings), the lubrication regime transitions from hydrodynamic to boundary lubrication — a critical failure mode that the Tolerance Agent can predict by tracking the continuous decrease of $d_{min}$ toward $h_{min}$ via the Non-Euclidean Wear model. The silence of the gap is loud with physics.[^10][^22]
