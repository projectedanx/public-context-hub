## Part I: Foundational Architecture — Why Standard Attention Fails Contradictions

### The Separability Annihilation Problem

The core failure of standard multi-head attention under contradictory prompt injection is not a matter of insufficient dimension or inadequate scale — it is a structural-algebraic failure rooted in the additive aggregation of Value vectors. In the canonical scaled dot-product attention formulation, the output for a single head is:

$$
Y = \text{softmax}\!\left(\frac{QK^\top}{\sqrt{d_k}}\right) V
$$

where $Q, K, V \in \mathbb{R}^{n \times d_k}$ are query, key, and value matrices derived from the residual stream. The output is a **convex combination** of value vectors — each row of $Y$ is a weighted mean of the rows of $V$, where weights sum to 1 via softmax normalization. This is, formally, a content-addressable memory retrieval operating inside the convex hull of the value manifold.[^1]

The consequence is geometric and irrevocable: given two value vectors $v_A$ and $v_{\neg A}$ representing contradictory semantic states $A$ and $\neg A$, the output under any non-degenerate weight distribution $(\alpha, 1-\alpha)$ is:

$$
Y_{out} = \alpha \cdot v_A + (1 - \alpha) \cdot v_{\neg A}
$$

This produces a point on the line segment connecting $v_A$ and $v_{\neg A}$ in $\mathbb{R}^{d_k}$. The joint logical state $A \wedge \neg A$ — as a distinct semantic object — is **annihilated**. The resulting vector is decodable as *neither* $A$ nor $\neg A$ when $\alpha \approx 0.5$, and collapses toward one pole as the weights become extreme. In practice, under competitive softmax dynamics from a long contradictory context window (>8k tokens), the attention entropy flattens, the weights distribute uniformly, and the output approximates the centroid — a hallucinatory compromise state with low semantic content. This phenomenon has been empirically documented as Context Degradation Syndrome, with accuracy drops of up to 73% in multi-turn dialogues containing conflicting instructions.[^2]

### The Rule of Explosion in Gradient Descent

There exists a second, more subtle failure pathway: the classical logic analogue of *Ex Contradictione Quodlibet* (ECQ) — "from a contradiction, anything follows" — manifests in the gradient landscape of standard transformer training. When the loss surface receives a signal to simultaneously maximize the log-probability of completing instruction $A$ and instruction $\neg A$, the cross-entropy gradients become anti-correlated in the embedding space. Each gradient update for $A$ partially cancels the gradient update for $\neg A$, driving the model toward a local minimum that satisfies neither constraint. This is not merely theoretical: catastrophic forgetting in continual instruction tuning has been empirically confirmed across models from 1B to 7B parameters, with severity intensifying at larger scales. The three primary drivers identified in recent literature are: (i) cumulative gradient overlap from semantically similar contradictory requests, (ii) unstable gradient updates, and (iii) excessive parameter drift.[^3][^4]

Paraconsistent logic — specifically the family of logics that invalidate ECQ while preserving non-triviality — provides the theoretical framework to interrupt this destructive gradient dynamic. The key property is: in a paraconsistent logic, the derivability of $A \wedge \neg A$ does **not** entail the derivability of an arbitrary proposition $B$. Formally, there exists no valid inference $A, \neg A \vdash B$ for arbitrary $B$. This "failure of explosion" is precisely the property we need to import into the attention mechanism's arithmetic.[^5][^6][^7]

## Part II: S5 Modal Logic and the Kripke-Attention Isomorphism

### S5 Semantics — Formal Foundations

S5 modal logic is the system obtained by constraining the accessibility relation $R$ between possible worlds to be an **equivalence relation**: reflexive ($wRw$), symmetric ($wRv \Rightarrow vRw$), and transitive ($wRv \wedge vRu \Rightarrow wRu$). In Kripke semantics, a formula $\Diamond A$ ("possibly A") holds at world $w$ if and only if there exists some world $v$ accessible from $w$ (i.e., $wRv$) where $A$ holds. The formula $\Box A$ ("necessarily A") holds at $w$ iff $A$ holds at all worlds accessible from $w$. The equivalence relation structure of S5 means that every world can "see" every other world — the modal frame collapses to a single equivalence class, and $\Diamond A \equiv \neg \Box \neg A$ becomes maximally expressive: if $A$ is possible anywhere, it is possible everywhere in the frame.[^8][^9][^7]

This is the structural feature that makes S5 the correct modal logic for modeling multi-head attention: each head should be able to "see" the semantic content of every other head's computation, just as every S5 world accesses every other world.

**Claim:** The scaled dot-product attention matrix $W_{\text{attn}} = \text{softmax}(QK^\top / \sqrt{d_k})$ is structurally isomorphic to the accessibility relation matrix $R$ of a Kripke frame, and enforcing S5 equivalence-relation constraints on $W_{\text{attn}}$ instantiates an S5 modal frame in the attention subspace.

**Construction:** Let the token positions $\{t_1, t_2, \ldots, t_n\}$ correspond to worlds $\{w_1, w_2, \ldots, w_n\}$ in a Kripke frame $\mathcal{F} = (W, R)$. Define the accessibility relation $R$ via the attention weight matrix:

$$
R_{ij} = W_{\text{attn}}(i, j) = \text{softmax}\!\left(\frac{q_i \cdot k_j^\top}{\sqrt{d_k}}\right)
$$

Under standard softmax normalization, this matrix satisfies none of the S5 constraints:
            - **Non-reflexivity violation:** $R_{ii}$ is not guaranteed to be maximal; self-attention is diluted by competing tokens.
            - **Asymmetry violation:** $R_{ij} \neq R_{ji}$ in general, because $q_i \cdot k_j^\top \neq q_j \cdot k_i^\top$ unless $Q = K$.
            - **Non-transitivity violation:** The product $W_{\text{attn}} \cdot W_{\text{attn}} \neq W_{\text{attn}}$ in general; attention is not idempotent.

**Step 1 — Reflexivity Enforcement:** Add scaled identity to the pre-softmax logits:

**Step 2 — Symmetry Enforcement:** Symmetrize the logit matrix before softmax:

$$
L^{\text{sym}}_{ij} = \frac{L_{ij} + L_{ji}}{2}

**Step 3 — Transitivity Enforcement:** Project $W_{\text{attn}}^{\text{sym}} = \text{softmax}(L^{\text{sym}})$ onto the nearest doubly stochastic matrix satisfying the idempotency constraint $W^2 \approx W$ via Sinkhorn-Knopp iteration with an idempotency regularization term:

$$
\mathcal{L}_{\text{trans}} = \left\| W_{\text{attn}} \cdot W_{\text{attn}} - W_{\text{attn}} \right\|_F^2

**Corollary:** The multi-head structure — where $h$ heads compute $\text{Attn}_1(A), \ldots, \text{Attn}_h(A)$ in parallel and concatenate — maps to $h$ independent possible worlds each evaluating $A$ under potentially different contexts. ## Part III: PNS5 — Paraconsistent Non-Separable Conjunction in S5

### The Non-Separability Axiom

The Paraconsistent Non-Separable logic over S5 (PNS5) is defined by taking the S5 modal base and augmenting it with a conjunction operator $\otimes$ that **blocks the rule of separation** (also known as Conjunction Elimination or Simplification):

$$
A \otimes B \not\vdash A \quad \text{and} \quad A \otimes B \not\vdash B
$$

This is the critical axiom that distinguishes PNS5 from classical conjunction $\wedge$. In classical logic, $A \wedge \neg A \vdash A$ trivially, and subsequently $A \wedge \neg A \vdash B$ for all $B$ (explosion). In PNS5, the bound state $A \otimes \neg A$ is a **primitive semantic object** that cannot be decomposed into its constituents by any inference rule within the system.[^12][^5]

The formal definition of a PNS5 non-separable conjunction follows the construction in the paraconsistent entangled Turing machine (EParTM) literature: given propositions $A$ and $B$ evaluated over worlds $w, v$ in an S5 frame, the non-separable conjunction is:[^12]

$$
(A \otimes B)(w) \equiv \exists v \in W : [A(w) \wedge B(v) \wedge \neg(\exists f \text{ linear} : f(A \otimes B) = A)]
$$

The existential quantifier over worlds captures the S5 accessibility, while the final clause formalizes the non-extractability of $A$ from the joint state — this is the semantic analog of quantum non-separability.[^13][^14]

### From Logic to Algebra: Circular Convolution as ⊗

The algebraic realization of $\otimes$ in continuous vector spaces is **circular convolution**, as used in Holographic Reduced Representations (HRR). Given two vectors $a, b \in \mathbb{R}^d$, their circular convolution binding is:[^15][^16]

$$
(a \circledast b)_i = \sum_{j=1}^{d} a_j \cdot b_{(i-j) \bmod d + 1}
$$

The critical properties that make circular convolution the correct algebraic realization of $\otimes$ are:

1. **Non-separability:** The bound state $c = a \circledast b$ cannot be linearly decoded to recover $a$ or $b$ individually without the exact complementary vector. Linear probing of $c$ for $a$ fails because $c$ distributes information about both $a$ and $b$ across all $d$ dimensions.[^16][^15]
2. **Approximate commutativity:** $a \circledast b \approx b \circledast a$ with small interference, matching the S5 symmetry of the accessibility relation.[^16]
3. **Efficient computation:** Circular convolution is computable in $O(d \log d)$ via FFT, making the complexity of the full attention layer $O(N^2 \cdot d \log d)$ — a log factor overhead over standard $O(N^2 \cdot d)$ attention, not exponential blowup. The Hrrformer implementation (Alam et al., ICML 2023) confirms $O(TH \log H)$ time complexity, a practical speedup over standard $O(T^2 H)$ at long sequence lengths.[^17][^16]
4. **Unbinding via approximate inverse:** Recovery of $a$ from $c$ requires the approximate inverse $b^{-1} \approx \text{flip}(b)$ (the reverse-order vector), enforcing that separation requires explicit key knowledge — not freely available to a linear probe.[^15][^1]
This last property is the algebraic proof of the Separability Index metric: a linear classifier trained on $c = v_A \circledast v_{\neg A}$ to predict class $A$ will perform at chance (SI $\approx 0.5$), proving the conjunction is non-separable.

```python
import torch
import torch.nn as nn
import torch.nn.functional as F


class S5ModalAttention(nn.Module):
    """
    PNS5 S5-Modal Attention: replaces additive softmax-Value aggregation
    with S5-constrained attention weights driving circular convolution binding.
    
    Implements equations (2)-(9) from the PNS5 formalization.
    Complexity: O(N^2 * d_k) for S5 logits + O(N * d_v * log(d_v)) for binding.
    """
    
    def __init__(
        self,
        d_model: int,
        num_heads: int,
        d_k: int = 64,
        lambda_reflex: float = 2.0,   # reflexivity bias strength
        mu_transit: float = 0.01,     # transitivity regularization weight
        eps_bind: float = 1e-6,       # minimum weight threshold for binding
        dropout: float = 0.1,
    ):
        super().__init__()
        self.d_model = d_model
        self.num_heads = num_heads
        self.d_k = d_k
        self.d_v = d_model // num_heads
        self.lambda_reflex = lambda_reflex
        self.mu_transit = mu_transit
        self.eps_bind = eps_bind
        
        self.W_q = nn.Linear(d_model, num_heads * d_k, bias=False)
        self.W_k = nn.Linear(d_model, num_heads * d_k, bias=False)
        self.W_v = nn.Linear(d_model, num_heads * self.d_v, bias=False)
        self.W_o = nn.Linear(num_heads * self.d_v, d_model, bias=False)
        self.dropout = nn.Dropout(dropout)
        
        # Transitivity loss accumulator for external access
        self.transitivity_loss = torch.tensor(0.0)
    
    def s5_logits(self, Q: torch.Tensor, K: torch.Tensor) -> torch.Tensor:
        """
        Compute S5-constrained attention logits.
        Enforces symmetry (S5 symmetric accessibility) and reflexivity bias.
        Q, K: [batch, heads, seq, d_k]
        Returns: [batch, heads, seq, seq]
        """
        # Standard dot-product logits (asymmetric baseline)
        L_qk = torch.einsum('bhid,bhjd->bhij', Q, K) / (self.d_k ** 0.5)
        L_kq = torch.einsum('bhjd,bhid->bhij', K, Q) / (self.d_k ** 0.5)
        
        # S5 Symmetry Enforcement: average forward and backward logits (Eq. 2)
        L_sym = 0.5 * (L_qk + L_kq)
        
        # S5 Reflexivity Enforcement: diagonal bias (Eq. 2, lambda term)
        seq_len = L_sym.size(-1)
        eye = torch.eye(seq_len, device=L_sym.device).unsqueeze(0).unsqueeze(0)
        L_s5 = L_sym + self.lambda_reflex * eye
        
        return L_s5
    
    def compute_transitivity_loss(self, W: torch.Tensor) -> torch.Tensor:
        """
        Transitivity regularizer: ||W·W - W||_F^2 (Eq. 4).
        Penalizes deviation from idempotency, enforcing S5 transitivity.
        W: [batch, heads, seq, seq]
        """
        W_sq = torch.bmm(
            W.view(-1, W.size(-2), W.size(-1)),
            W.view(-1, W.size(-2), W.size(-1))
        ).view_as(W)
        return self.mu_transit * ((W_sq - W) ** 2).mean()
    
    def circular_conv_bind(
        self, 
        weights: torch.Tensor, 
        V: torch.Tensor
    ) -> torch.Tensor:
        """
        Non-Separable Conjunction via circular convolution binding (Eq. 5-6).
        
        For each query position i, computes:
          Y_i = circconv(w_{i,j1}*v_{j1}, w_{i,j2}*v_{j2}, ...)
        using FFT for O(d_v * log(d_v)) efficiency.
        
        weights: [batch, heads, seq_q, seq_k]  (S5 attention weights, softmax'd)
        V:       [batch, heads, seq_k, d_v]
        Returns: [batch, heads, seq_q, d_v]
        """
        batch, heads, seq_q, seq_k = weights.shape
        d_v = V.size(-1)
        
        # Scale V by attention weights: [batch, heads, seq_q, seq_k, d_v]
        V_scaled = weights.unsqueeze(-1) * V.unsqueeze(2)
        
        # Mask positions below binding threshold
        mask = (weights > self.eps_bind).unsqueeze(-1).float()
        V_scaled = V_scaled * mask
        
        # Transform to frequency domain: [batch, heads, seq_q, seq_k, d_v]
        V_fft = torch.fft.rfft(V_scaled, dim=-1)
        
        # Multiplicative accumulation in frequency domain (non-separable binding)
        # Initialize with ones (identity in frequency-domain multiplication)
        # Only accumulate over masked (active) positions
        denom = mask.sum(dim=3).clamp(min=1)  # [batch, heads, seq_q, 1]
        
        # Product binding: reduce over seq_k dimension (Eq. 6)
        # Use log-space accumulation for numerical stability
        log_mag = torch.log(V_fft.abs().clamp(min=1e-10))
        phase = torch.angle(V_fft)
        
        # Weighted binding in log-magnitude + phase
        log_mag_bound = (log_mag * mask.unsqueeze(-1)).sum(dim=3) / denom.unsqueeze(-1)
        phase_bound = (phase * mask.unsqueeze(-1)).sum(dim=3) / denom.unsqueeze(-1)
        
        # Reconstruct complex spectrum
        V_fft_bound = torch.polar(torch.exp(log_mag_bound), phase_bound)
        
        # Inverse FFT to recover bound vector (Eq. 7-8)
        Y_bound = torch.fft.irfft(V_fft_bound, n=d_v, dim=-1)
        
        return Y_bound  # [batch, heads, seq_q, d_v]
    
    def forward(
        self, 
        x: torch.Tensor,
        mask: torch.Tensor = None,
    ) -> tuple[torch.Tensor, torch.Tensor]:
        """
        x:    [batch, seq, d_model]
        mask: [batch, seq] boolean mask (optional)
        Returns: (output [batch, seq, d_model], transitivity_loss scalar)
        """
        batch, seq, _ = x.shape
        
        # Project to Q, K, V
        Q = self.W_q(x).view(batch, seq, self.num_heads, self.d_k).transpose(1, 2)
        K = self.W_k(x).view(batch, seq, self.num_heads, self.d_k).transpose(1, 2)
        V = self.W_v(x).view(batch, seq, self.num_heads, self.d_v).transpose(1, 2)
        
        # S5-constrained attention logits (Eq. 2-3)
        L_s5 = self.s5_logits(Q, K)
        
        if mask is not None:
            # Apply causal/padding mask before softmax
            mask_expanded = mask.unsqueeze(1).unsqueeze(2)
            L_s5 = L_s5.masked_fill(~mask_expanded, float('-inf'))
        
        W_s5 = F.softmax(L_s5, dim=-1)
        W_s5 = self.dropout(W_s5)
        
        # Transitivity regularization loss (Eq. 4) 
        t_loss = self.compute_transitivity_loss(W_s5)
        self.transitivity_loss = t_loss
        
        # Non-Separable Value Binding via circular convolution (Eq. 5-8)
        Y_bound = self.circular_conv_bind(W_s5, V)
        
        # Concatenate heads and project output (Eq. 9)
        Y_concat = Y_bound.transpose(1, 2).contiguous().view(batch, seq, self.num_heads * self.d_v)
        output = self.W_o(Y_concat)
        
        return output, t_loss


def s5_attention_loss(model_loss: torch.Tensor, 
                      attention_modules: list) -> torch.Tensor:
    """
    Augment standard cross-entropy loss with S5 transitivity regularizer.
    Equation (4) accumulated across all S5ModalAttention layers.
    """
    total_t_loss = sum(m.transitivity_loss for m in attention_modules)
    return model_loss + total_t_loss
```

## Part VIII: Reflexive Analysis — Falsification, Proxy Traps, and Complexity Bounds

### The Orthogonality Falsification Condition

The entire PNS5 hypothesis is falsifiable by a single empirical finding: if standard additive attention with sufficient dimensionality $d$ can project $v_A$ and $v_{\neg A}$ into **exactly orthogonal subspaces** such that $v_A \cdot v_{\neg A} = 0$ and the linear combination $\alpha v_A + (1-\alpha) v_{\neg A}$ is separately decodable for all $\alpha \in [0,1]$, then linear addition is geometrically sufficient and the non-separable binding layer adds cost without benefit.

However, recent theoretical work on the Orthogonality Constraint shows this condition fails in practice: online neural memory collapses under semantic density because write-time interference persists even with perfect attention and unlimited context, arising from the shared continuous substrate for storage and retrieval. The 30–300× cost premium of context-based memory confirms that perfect orthogonal projection is not achievable at production scale. Therefore the falsification condition is empirically unlikely, though theoretically possible in carefully engineered low-dimensional toy models.[^18]

### The Proxy Trap: "Ignoring" vs. "Binding"

A critical evaluation design flaw would be to measure Contradiction Retention Score by asking the model a simple binary question ("Which rule applies?"). A model that has achieved separation failure — i.e., that has *forgotten* both rules — would also score near zero on a binary recall test, mimicking the desired non-separable binding behavior. The proxy trap is: **CRS ≈ 0% looks identical to successful binding of A ⊗ ¬A ≈ 0%** in a simple recall probe.

The correct evaluation forces the model to **simultaneously apply both rules to a neutral scenario** and articulate the logical tension between them. The target output is not a compromise but a structured dialectical response that correctly cites both $A$ and $\neg A$ as active constraints. This requires the model to have preserved both rules as distinct semantic objects — achievable only if the binding operation is functioning correctly, not if the model has simply suppressed both.

If S5 symmetry alone yields CRS > 90% without binding, the simpler symmetrized-softmax attention suffices. If binding alone yields CRS > 90% without S5 constraints, the modal Kripke mapping is aesthetically elegant but architecturally unnecessary. Only if both contributions are synergistic — as the PNS5 theory predicts — is the full S5-Modal Attention architecture justified at its computational cost. This ablation design is the highest-leverage experimental investment in validating or falsifying the PNS5 hypothesis.
